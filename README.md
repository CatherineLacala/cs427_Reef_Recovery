# 🪸Reef_Recovery
This original team project was created for our CS427 Creative Coding class.\
Within this course, we spent the entire semester creating a 3D VR experience using the 2019 Unity version to be presented at the UIC CAVE2.\
\
GOAL: Our project aims to visualize coral reef health as a dynamic system affected by human intervention. Instead of focusing solely on reef destruction, the project has users apply different restoration strategies which affect the reef’s appearance.

This project helps advance research because it translates complicated marine ecology into experiential understanding as well as providing insight to how people understand ecosystem resilience when changes are experienced. It also supports previous research about how immersive visualizations improve comprehension of long-term environmental processes

Reef Recovery contributes to the Sustainable Development Goal #14: Life Below Water, where it shows cause-and-effect relationships between human behavior and reef health.
